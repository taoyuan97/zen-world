# 星野 · 星空原对话剧本（定稿）

- 决策 M3-D1：2~4 轮 + 1 组选项 + `start_meditation` 收尾；星野口吻：诗意、辽阔，善用星夜意象（GDD §4「星野诗意」）
- 对应数据：`src/data/dialogues/starfield.json`

## 流程

1. **greeting**：你看，星星不说话，却一直在回答。 → q1
2. **q1（选项）**：抬起头的时候，你觉得自己是大，还是小？
   - 选项 A「小，像一粒尘埃」→ r1
   - 选项 B「大，像能装下整片夜空」→ r2
3. **r1**：尘埃也有星光落在上面。小，不等于微不足道。 → invite
4. **r2**：那就对了。能装下夜空的，本来就是你自己。 → invite
5. **invite**：坐好吧。让星星陪你，把这一口气，吐得很长很长。 → action: `start_meditation`

## 完成页赠言（blessing）

> 今夜你坐过的这片星空，会一直为你留着。
