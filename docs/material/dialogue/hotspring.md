# 暖阳 · 温泉谷对话剧本（定稿）

- 决策 D4：2~4 轮 + 1 组选项 + `start_meditation` 收尾；暖阳口吻：热情绵软、爱用叠词与问候，口头禅「暖暖地」
- 对应数据：`src/data/dialogues/hotspring.json`

## 流程

1. **greeting**：哎呀，可把你盼来了！快坐，快坐。先把手伸到热气上烘一烘，暖暖地。这一路走来，身体累了吧？ → q1
2. **q1（选项）**：你知道吗，身体记得一切。你扛过的累，它都替你收着。今天，它最想跟你说句什么？
   - 选项 A「它想说：我累了」→ r1
   - 选项 B「它想说：我还撑得住」→ r2
3. **r1**：哎，听见就好，听见就好。累了就靠过来，让热气把肩膀上的紧，一点一点烫开。 → invite
4. **r2**：这孩子，就是太懂事。越懂事，越要疼。来，让泉水替你撑一会儿。 → invite
5. **invite**：来，坐近些。我们从头顶到脚尖，把身体的每一寸，都暖暖地问候一遍。 → action: `start_meditation`

## 完成页赠言（blessing）

> 泉水记下了你的暖，也替你收好了这一路的累。
