# 了然 · 古寺对话剧本（定稿）

- 决策 D4：2~4 轮 + 1 组选项 + `start_meditation` 收尾；了然口吻：缓慢郑重、句句留白，口头禅「好，好。」；按终章语气撰写，像送别又像托付
- 对应数据：`src/data/dialogues/temple.json`

## 流程

1. **greeting**：你来了。钟声刚落，灯火正好。这一路的山，你一座一座，都走过来了。 → q1
2. **q1（选项）**：寺里的灯，从来不是一盏点亮的。是一盏，点亮另一盏。今天，你想把这盏灯，递给谁？
   - 选项 A「递给一位我惦念的人」→ r1
   - 选项 B「递给那个一直撑着的自己」→ r2
3. **r1**：好，好。心里装着他的样子，祝福就已经上路了。 → invite
4. **r2**：好，好。走了这么远，最该被照见的，是自己。 → invite
5. **invite**：来吧，坐到灯前来。钟声再起的时候，我们一起，把慈悲先给自己，再给众生。 → action: `start_meditation`

## 完成页赠言（blessing）

> 灯灯相传。你坐过的这盏灯，会一直亮着——照你下山，也照后来的人上山。
