# 涟漪 · 镜湖畔对话剧本（定稿）

- 决策 D4：2~4 轮 + 1 组选项 + `start_meditation` 收尾；涟漪口吻：柔和、有流动感，以水喻情绪，口头禅「会平的」
- 对应数据：`src/data/dialogues/lakeside.json`

## 流程

1. **greeting**：你来了。坐近一点。湖水很轻，它听得见你心里的声音。 → q1
2. **q1（选项）**：此刻，你心里的水面，是什么样子？
   - 选项 A「起了好大的浪」→ r1
   - 选项 B「有一层细细的波纹」→ r2
3. **r1**：会平的。水从不跟浪生气——浪来了，浪也会走，水一直都在。 → invite
4. **r2**：会平的。波纹是风路过的地方。不用抚平，看着它一圈一圈散开，就好。 → invite
5. **invite**：来，到湖边坐下。我们什么都不做，陪水面慢慢静下来。 → action: `start_meditation`

## 完成页赠言（blessing）

> 湖水把这一段安静，折成了一圈很小的涟漪。
