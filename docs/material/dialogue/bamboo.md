## 流程

1. **greeting**：你来了。竹子不说话的时候，其实是在听风。 → q1
2. **q1（选项）**：今天，你的心像什么？
   - 选项 A「像被风吹乱的竹叶」→ r1
   - 选项 B「像一潭静水」→ r2
3. **r1**：那正好。风停不停不重要，我们只需要看见叶子在动。 → invite
4. **r2**：静水也会照见云影。坐一会儿，看看它会照见什么。 → invite
5. **invite**：来吧，坐下来。我们一起，只关注这一口呼吸。 → action: `start_meditation`
