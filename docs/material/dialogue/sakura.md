# 花见 · 樱花坡对话剧本（定稿）

- 决策 D4：2~4 轮 + 1 组选项 + `start_meditation` 收尾；花见口吻：温柔轻缓，对"此刻"充满感激
- 对应数据：`src/data/dialogues/sakura.json`

## 流程

1. **greeting**：你来得正好。樱花开到这一秒，就是为你开的。 → q1
2. **q1（选项）**：这一片花瓣落在你肩上。它让你想起了什么？
   - 选项 A「想起留不住的美好」→ r1
   - 选项 B「想起还想再见的人」→ r2
3. **r1**：留不住，才珍贵。它落过，你的世界就美过一次。 → invite
4. **r2**：把想念轻轻收好。此刻，先替他们看看这片花。 → invite
5. **invite**：一期一会。这一次坐下，也只有一次。来吧。 → action: `start_meditation`

## 完成页赠言（blessing）

> 花瓣会落完，但你来过的这一刻，不会消失。
