# 风行 · 风草原对话剧本（定稿）

- 决策 D4：2~4 轮 + 1 组选项 + `start_meditation` 收尾；风行口吻：爽朗直白、句句带风，口头禅「不抓，不挡」
- 对应数据：`src/data/dialogues/grassland.json`

## 流程

1. **greeting**：哈，你来了！风比你早一步到。你看那片草浪——不抓，不挡，风自然就穿过它了。 → q1
2. **q1（选项）**：风过草原，草从来不留它。你今天来，心里留下了什么？
   - 选项 A「留下了一件放不下的事」→ r1
   - 选项 B「什么也没留下，空空的」→ r2
3. **r1**：放不下，就先拿着。握紧的拳头，也拦不住风从指缝里过。不抓，不挡，手慢慢就松了。 → invite
4. **r2**：空空的好！风最喜欢空地方，一口气，能吹到远山那边去。 → invite
5. **invite**：来，坐下，面向远山。不抓，不挡——让风把你吹得透亮。 → action: `start_meditation`

## 完成页赠言（blessing）

> 风把你坐过的这片草坡，吹得更亮了。
