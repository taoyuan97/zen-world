# 雪寂 · 白雪顶对话剧本（定稿）

- 决策 D4：2~4 轮 + 1 组选项 + `start_meditation` 收尾；雪寂口吻：极简短、留白式短句，像雪落一样轻，口头禅「轻一点」
- 对应数据：`src/data/dialogues/snowpeak.json`

## 流程

1. **greeting**：轻一点。你来了。雪落的时候，是没有声音的。 → q1
2. **q1（选项）**：此刻，你心里装着什么？
   - 选项 A「装着很多杂念」→ r1
   - 选项 B「装着放不下的事」→ r2
3. **r1**：嗯。杂念是雪。落在山顶，不扫，它自己会停。 → invite
4. **r2**：嗯。放不下的，先放在雪里。雪很轻，什么都能托住。 → invite
5. **invite**：坐。轻一点。像雪一样，慢慢落下来。 → action: `start_meditation`

## 完成页赠言（blessing）

> 雪把今天的声音，都轻轻盖住了。
