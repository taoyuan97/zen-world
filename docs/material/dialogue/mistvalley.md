# 雾隐 · 雾谷对话剧本（定稿）

- 决策 M3-D1：2~4 轮 + 1 组选项 + `start_meditation` 收尾；雾隐口吻：慢条斯理、安抚，与雾共处
- 对应数据：`src/data/dialogues/mistvalley.json`

## 流程

1. **greeting**：不急。雾里走路，快也没有用。 → q1
2. **q1（选项）**：你现在的心里，有没有看不清的事？
   - 选项 A「有，看不清前面的路」→ r1
   - 选项 B「有，看不清自己的心」→ r2
3. **r1**：看不清路，就看清脚下这一步。一步，就够了。 → invite
4. **r2**：心不急着看清。雾里的山，也还是山。 → invite
5. **invite**：坐吧。我们不散雾，我们陪雾坐一会儿。 → action: `start_meditation`

## 完成页赠言（blessing）

> 雾没有散，但你已经不怕它了。
