## 1. 主闭环（对应任务文档 §2 验收标准第 1 条）

按顺序走一遍完整流程，每步检查右侧表现：

| 步骤 | 操作 | 预期表现 |
|---|---|---|
| 1 | 地图点击青竹林山体 | 过渡遮罩淡出，进入冥想场景，加载进度可见 |
| 2 | 入场运镜 | 相机 2.5s 缓推（cubic-out），期间点击/键盘无效 |
| 3 | 场景观感 | 圆形地块（直径约 40m）、day 天穹、程序化竹林、落叶粒子、雾色与主题配色一致；老师坐于蒲团上（下半身被遮挡） |
| 4 | 悬停老师 | 提示"与老师交谈"；老师有待机呼吸/随机点头动作 |
| 5 | 点击老师 | 切对话过肩机位，对话框出现 |
| 6 | 对话 | 打字机 20~30 字/秒；点击立即补全；Space/Enter 前进；出现 1 组选项，选不同 reply 有对应回应；竹心口吻符合 GDD 文案规范 |
| 7 | 对话收尾 | 弹出时长选择面板（5 分钟 / 10 分钟） |
| 8 | 选 5 分钟 | 切冥想机位（缓慢环绕 2°/s）；HUD 出现：SVG 圆环、剩余分钟数、呼吸环（吸 4s / 呼 4s）、引导语区 |
| 9 | 冥想中 | 引导语按 cue 时间淡入淡出（对照 GDD 附录 B 5 分钟定稿）；老师动作幅度减半 |
| 10 | 计时结束 | 钵音 → 光柱升起 → 金粒扩散 → 场景配色 unlit→lit 渐变 → 光柱淡出（约 4s）；演出期间点击/键盘无效 |
| 11 | 完成页 | 显示本次时长、累计次数、累计时长、竹心赠言、返回地图按钮 |
| 12 | 返回地图 | 青竹林山体变为点亮态（顶部灯亮、材质变亮），HUD 进度 1/10 |

## 2. 10 分钟档（第 2 条）

- [ ] 带 `?debug` 打开，使用调试加速计时（加速仅限调试模式）走通 10 分钟版
- [ ] 引导词为 10 分钟版（同结构拉长，cue 升序、最大 t < duration）
- [ ] 不带 `?debug` 时不出现加速入口

## 3. 中断与恢复（第 3、4 条）

- [ ] 冥想中切到别的标签页：自动暂停，HUD 提示"已为你暂停"
- [ ] 切回后手动恢复：剩余时间准确（与墙上时钟对照，误差 < 2s）
- [ ] 冥想中按 Esc / 返回：弹确认框；确认退出 → 本次不计完成，回地图山仍未点亮；取消 → 继续冥想

## 4. 重复冥想与累计（第 5 条）

- [ ] 已点亮后再次进入青竹林：可完整再走一遍
- [ ] 第二次完成页：累计次数 = 2，累计时长 = 两次之和
- [ ] 刷新页面后累计数据保留（localStorage `zen-world-save-v1`）

## 5. 音频与静音（第 6 条）

- [ ] 首次交互后，点亮演出钵音可闻（振荡器合成，约 1.5s 衰减，不刺耳即可）
- [ ] UI 点击音可闻
- [ ] HUD 常驻静音按钮：点击即时静音/恢复；刷新页面后静音状态保留

## 6. 稳定性（第 7 条）

- [ ] 演出期间输入被吞掉（点击、Esc 均无效）
- [ ] `?debug` 打开，地图 ↔ 冥想场景往返 10 次后，`renderer.info` 的 geometries / textures 数量回到基线（决策 D7 口径）
- [ ] 控制台无红色报错（资源加载失败只允许 console.warn + 程序化占位回退）

## 7. 构建（第 8 条）

- [ ] `npm run build` 通过（tsc --noEmit 零错误 + vite 打包成功）
