## 1. 十山遍历（对应任务文档 §2 验收标准第 1 条）

每座山走一遍：进山 → 点老师对话（过选项）→ 选 5 分钟（debug 加速）→ 点亮演出 → 完成页 → 回地图点亮。每山核对下表专属表现：

| 山 | 天空 | 装饰/粒子专属表现 | 老师配饰 | 对话主题 |
|---|---|---|---|---|
| bamboo 青竹林 | day | 竹丛+石灯笼、落叶（M2 已验） | 竹杖 | 第一次呼吸 |
| snowpeak 白雪顶 | day | 松树+岩石、**雪花飘落** | 念珠 | 雪落无声 |
| lakeside 镜湖畔 | day | 芦苇+石灯笼+**水面反光片**（微光呼吸）、**萤火漂浮** | 茶碗 | 水知道答案 |
| starfield 星空原 | **night** | **星空穹顶（星点+流星）**+岩石+微光水晶、萤火 | 星图 | 你比想象辽阔 |
| desert 寂静沙漠 | **sunset** | 枯树+岩石+**沙丘弧线**、无粒子 | 木鱼 | 少即是多 |
| sakura 樱花坡 | day | 樱花树+石灯笼、**花瓣飘落** | 纸伞 | 一期一会 |
| mistvalley 雾谷 | day | 松树+岩石、**浓雾**（能见度明显低于他山） | 拐杖 | 看不清也没关系 |
| grassland 风草原 | day | **草浪摇摆**+岩石、落叶 | 芦笛 | 让风穿过你 |
| hotspring 温泉谷 | sunset | 岩石+蒸汽孔、**蒸汽上升**、**呼吸环 5s/5s** | 木勺 | 身体记得一切 |
| temple 古寺 | sunset | **寺塔×2+幡（轻摆）**、萤火 | 寺钟 | 灯灯相传（终章语气） |

每山通用项：

- [ ] 入场运镜 → 悬停提示 → 对话（打字机/选项分支正常）→ 时长面板 → 冥想 HUD 正常
- [ ] 引导语按 cue 出现、与该山引导词 md 一致；呼吸环节奏正确（温泉谷 5s/5s，其余 4s/4s）
- [ ] 计时结束 → 演出 → 完成页（本次时长/累计/该山 blessing 赠言）→ 回地图该山点亮
- [ ] 10 分钟档（抽查 2~3 山即可）走通
- [ ] console 无报错（资源 warn 回退除外）

## 2. 视觉差异（第 2 条）

- [ ] 9 个新主题与他山并置差异明显：配色、装饰、天空时段（day/night/sunset）、粒子均按配置生效
- [ ] night（星空原）可见星点与流星；sunset（沙漠/温泉谷/古寺）为天边暖色渐变
- [ ] 雾谷浓雾明显比他山能见度低（fogNear/fogFar=9/46）

## 3. 老师差异化（第 3 条）

- [ ] 10 位老师并置可区分：衣色不同 + 配饰不同（竹杖/念珠/茶碗/星图/木鱼/纸伞/拐杖/芦笛/木勺/寺钟）
- [ ] 待机呼吸/随机点头正常；冥想中动作幅度减半

## 4. 数据健壮性（第 4 条）

- [ ] 全量校验通过（已自动完成：30/30，见遍历记录）
- [ ] 任选一份引导 JSON 临时改坏（如删 cue），该山提示"维护中"且不影响其他山；改回后恢复

## 5. 点亮 10/10（第 5 条）

- [ ] 用 debug 点亮全部 10 山后回地图：各山灯色/萤火共存不杂乱，全景氛围符合预期
