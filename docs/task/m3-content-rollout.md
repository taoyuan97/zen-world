## 2. 验收标准

- [ ] 10 座山全部可完整走通：进山 → 对话 → 冥想（5 与 10 分钟两档）→ 点亮 → 回地图点亮
- [ ] 9 个新主题视觉差异明显：配色、装饰、天空时段（day/night/sunset）、粒子均按配置生效
- [ ] 10 位老师外观可区分（配色 + 配饰），待机动作正常
- [ ] 全部 JSON 通过校验器；任意文件损坏不影响其他山
- [ ] 点亮 10/10 后地图全景氛围符合预期（各山灯色/萤火共存不杂乱）
- [ ] 文案校对完成，无占位文本残留
- [ ] 性能复查：任一场景 60fps、draw calls < 80
- [ ] `npm run build` 通过

## 3. 风险与备注

- **工作量大头是文案**（3.4 + 3.5 约 4 人日起草 + 校对）：AI 起草后务必逐山朗读校对，尤其引导词的停顿节奏与 cue 时间轴匹配。
- 9 主题共用代码、只换配置；如发现某主题需要新代码（如水面反射），先评估能否用配置组合实现，避免破例。
- 星空原（night）与古寺（sunset）首次引入非 day 天空，skyDome 预设需在 3.2 一并落地。
