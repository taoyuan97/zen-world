## 3. 全局完成定义（DoD）

每个任务勾选前必须满足：

1. `npm run build`（含 `tsc --noEmit`）零错误零警告；
2. 桌面 Chrome / Edge 最新版手动验证通过；
3. 无 console 报错（资源加载失败的 warn 占位回退除外）；
4. 涉及数据的任务：校验器全量通过；
5. 里程碑验收：对应文档"验收标准"逐条在浏览器中实测通过。

## 4. 通用约定

- **数据驱动红线**：新增主题/文案只改 `src/data/`，不改场景代码（TDD §7 红线）。
- **调试开关**：`?debug` 性能面板 + 点亮状态调试；`?hill=<id>` 直进场景（M2 起）。
- **存档兼容**：任何涉及存档字段的任务必须验证旧档可正常读取（SaveSystem 迁移钩子）。
- **验收方式**：本地 `npm run dev` 浏览器手动验收，按各里程碑文档清单逐项打勾。
