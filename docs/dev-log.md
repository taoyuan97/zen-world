# 开发日志

## 2026-07-21

### Added

- 老师配置支持可选自定义 GLB 模型字段，用于按山指定独立老师形象。

### Changed

- 古寺场景老师模型替换为 huangniu-zen-master.glb，保持原始尺寸与纹理，去除蒲团并重新定位寺钟配饰。

## 2026-07-20（M4 体验打磨）

### Added

- 新增 Howler 音频管理器：10 山主题环境音循环渐起渐弱、200 条 TTS 引导语音与文字 cue 同步播放、交互音效分组与主静音统一控制、后台标签页音频暂停恢复。

- 新增 10/10 全局完成演出：地图全景灯光秀、终局贺词面板、古寺彩蛋对话与不限时自由冥想模式。

- 新增点亮山地图常驻萤火、光柱着色器流光微闪、环境粒子三级密度与移动端性能档位。

- 新增移动端基础适配：触摸手势、窄屏响应式布局与自动性能降级。

### Changed

- 点亮钵音与 UI 点击音由 WebAudio 合成占位替换为生成音频素材。

- 冥想 cue 时间轴按语音实际时长自动微调，语音缺失时回退纯文字。

### Removed

- 移除 WebAudio 占位音频系统（由 Howler 音频管理器替代）。

## 2026-07-20

### Fixed

- 修复冥想计时结束时完成页不触发的问题，并为事件总线与主循环添加防御性异常捕获。

### Added

- 新增 M2 青竹林垂直切片核心模块（教师模型、对话系统、冥想计时、三段式机位）。

- 新增竹心剧本、双版本引导词、占位音效系统与点亮演出特效。

- 新增冥想 HUD、收尾界面与端到端场景整合。

- 新增 9 座山主题场景（12 种程序化装饰、5 类差异化粒子、星空与流星、浓雾）与 10 位老师程序化配饰。

- 新增 9 山对话剧本与 5/10 分钟引导词各档文案（含 docs/material 双目录 md 存档），温泉谷呼吸节奏差异化 5s/5s。

- 新增 M3 十山遍历测试记录与人工验收文档。

### Changed

- M1 地图原型通过浏览器人工验收，同步更新项目状态。

- hills.json 补全水面反光、沙丘装饰项，并支持雾谷配置化浓雾（fogNear/fogFar）。

- M3 内容铺开通过浏览器人工验收（含文案集中校对），同步更新项目状态。
